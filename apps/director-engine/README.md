# Director Engine

Version: v0.1

Director Engine creates structured story beat and storyboard draft files from a project folder.

It is local-first and project-folder based.

## Usage

```bash
python apps/director-engine/create_storyboard.py \
  --project-folder projects/current
```

Output:

```text
projects/current/story-beats.md
projects/current/storyboard.md
```
