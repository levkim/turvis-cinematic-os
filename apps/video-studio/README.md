# TURVIS Video Studio

Version: v0.1

React GUI for creating fast video drafts.

## Run

```bash
cd apps/video-studio
npm install
npm run dev
```

Open the local app at:

```text
http://localhost:5173
```

## Current Features

- New Video Wizard
- Generic video categories
- Title, project ID, duration, and aspect ratio inputs
- Narration paste area
- CLI command guide for create, fast draft, and preview
